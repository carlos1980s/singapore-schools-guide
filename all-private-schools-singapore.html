Executive Summary

This report presents a comprehensive catalog of all private schools in Singapore, categorized using a custom classification system based on:

  • Affiliation Type (Religious, Ethnic, Independent, International)
  • Fee Structure Tiers (Budget <S$15K, Mid S$15-40K, Premium S$40-80K, Ultra-Premium >S$80K)
  • Curriculum Type (National, IB, British, Religious, Dual)
  • Language of Instruction (English, Chinese, Malay, Tamil, Arabic)
Key Finding: Fee range from S$3,000/year (madrasahs with subsidies) to S$70,000/year (premium international schools)

Classification System Used

Tier 1: By Institutional Affiliation

  1. International Schools - Foreign curriculum, diverse student body
  2. Religious Schools - Faith-based education (Islamic, Christian, Buddhist)
  3. Ethnic/Community Schools - SAP Chinese schools
  4. Independent Schools - Autonomous, often specialized
  5. Private Kindergartens - Early childhood education
